# Uptask

Order manager built with Next.js and Postgres

The administration route is at: `/admin/orders`

## Tech Stack

- Next.js
- Postgres
- Prisma

## Development

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/oscar370/quiosco_next_app_router.git
cd quiosco_next_app_router
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

4. Build for production:

```bash
pnpm build
```
